Hi, I'm Rhys Powell I have over a decades worth of experience in the industry mostly in a system administration background moving to now also doing development.

As a systems administrator I wrote a lot of code but mostly in languages such as shell, python and ruby but with the devops culture and also being a keen supporter of infrastructure as code I started writing more and more code and thats why you find me here today.

With python being my go to language I needed to improve my front end development exposure, Freecodecamp has done that for me, with self paced learning, a gradual introduction to the code, lots of others around to help and giving me the structure that I needed to be able to get from nothing to here. They have achieved what many other courses have failed to do!
